About MyBot

Project Overview

MyBot is a chess style imitation project built using PyTorch and Flask.

Instead of maximizing playing strength, the objective was to train a neural network that reproduces my own playing style from historical games.

The model was trained on approximately 610,000 positions extracted from my personal Lichess games.

Feature Vector

  • Piece placement (64 squares)
  • Move number
  • Castling rights
  • En-passant availability
  • Side to move
  • Piece counts
  • Legal move count
  • Check status
  • Attack map for every square

Model

  • Framework: PyTorch
  • Dataset: ~610k positions
  • Output Space: 4096 possible moves
  • Inference: Top-k prediction with legality filtering
  • Hybrid evaluation using Stockfish

MODEL vs OVERRIDE

Every move displayed during a game indicates whether it was played directly by the neural network or selected by Stockfish.

  • ✔ MODEL — Neural network move accepted.
  • ⚙ OVERRIDE — Neural network move rejected after Stockfish evaluation. Also when the game becomes too trivial (8 pawns up to be exact) or there is a forced mate stockfish is allowed to take over because the model migth drag the game unnecessarily.
  • ♔ ENDGAME — Pure Stockfish is used for simple endgames.
